# ShoppingCart
A Demo Shopping application for coding practices.

1. Added **API** call using **Alamofire** to fetch item list.
2. Added Item table view to use **UITableview** and show list of items.
3. Added Detail view to show item details.
4. Used **Auto layout** to handle diffrent device type and orientations.
5. Created an **Image cache framework** to chache image and created an **Swift package manager** for it.
6. Added **UICollectionView** and **UIPageControl** to show image slide. show of item detail.
